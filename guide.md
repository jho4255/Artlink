## 1. 시작하기 (공통)

### 1-1. 접속

- **배포 URL**: https://artlink-2esp.onrender.com
- **로컬 개발**: http://localhost:5173 (run_web.sh 실행 후)
- 모바일 브라우저에서도 최적화되어 있습니다 (PWA 지원).

### 1-2. 로그인

현재는 개발용 퀵 로그인 방식입니다.

1. 상단 네비게이션 바에서 **[마이페이지]** 를 클릭합니다.
2. 비로그인 상태이면 로그인 페이지로 이동합니다.
3. 4개의 테스트 계정 중 하나를 선택하면 즉시 로그인됩니다.

| 계정 | 역할 | 이메일 | 설명 |
|------|------|--------|------|
| Artist 1 | ARTIST | artist1@artlink.com | 포트폴리오 보유 |
| Artist 2 | ARTIST | artist2@artlink.com | 기본 아티스트 |
| Gallery Owner | GALLERY | gallery@artlink.com | 갤러리 3개 보유 |
| Admin | ADMIN | admin@artlink.com | 전체 운영 관리 |

### 1-3. 로그아웃

마이페이지 우측 상단의 빨간 **[로그아웃]** 버튼을 클릭합니다.

### 1-4. 네비게이션 바

모든 페이지 상단에 고정되어 있습니다.

- **좌측**: ArtLink 로고 (클릭 시 홈으로 이동)
- **우측 메뉴**: 홈, 갤러리, 모집공고, 혜택, 마이페이지
- **모바일**: 햄버거 메뉴(三)를 탭하면 메뉴가 펼쳐집니다.
- 현재 보고 있는 페이지가 하이라이트 표시됩니다.

## 2. 홈 화면 (공통)

### 2-1. 히어로 슬라이더

- 홈 화면 최상단에 큰 배너 이미지가 자동으로 슬라이드됩니다.
- **3초마다 자동 전환**됩니다.
- 좌우 화살표 버튼이나 터치 스와이프로 수동 전환할 수 있습니다.
- 수동 전환 시 3초 타이머가 리셋된 후 다시 자동 전환이 시작됩니다.
- **[바로가기]** 버튼이 있는 경우:
  - 외부 URL: 새 브라우저 탭에서 열림
  - 내부 URL: 해당 페이지로 이동

### 2-2. 캐치프레이즈

"갤러리와 아티스트를 잇다 : ArtLink" 문구가 중앙에 표시됩니다.

### 2-3. 퀵 액션 카드

3개의 바로가기 카드가 제공됩니다:
- **갤러리 찾기** -> 갤러리 목록 페이지
- **진행중인 공고** -> 모집 공고 페이지
- **혜택** -> 혜택 페이지

### 2-4. 이달의 갤러리

- Admin이 선정한 갤러리가 가로 스크롤 카드로 표시됩니다.
- 갤러리명, 별점, 대표 이미지를 확인할 수 있습니다.
- 카드 클릭 시 해당 갤러리 상세 페이지로 이동합니다.
- 등록 기한이 만료되면 자동으로 제거됩니다.

## 3. 갤러리 찾기 (공통)

### 3-1. 갤러리 목록

- 승인된 갤러리가 카드 형태로 나열됩니다.
- 각 카드에는 **대표 이미지, 갤러리명, 주소, 전화번호, 한줄 소개, 별점**이 표시됩니다.
- 카드를 클릭하면 갤러리 상세 페이지로 이동합니다.

### 3-2. 필터 사용법

1. 우측 상단 **[필터]** 버튼을 클릭하면 필터 패널이 열립니다.
2. 사용 가능한 필터:
   - **지역**: 서울, 경기 북부, 경기 남부, 대전, 부산 (하나만 선택 가능, 다시 클릭하면 해제)
   - **별점**: 3점 이상, 4점 이상 (하나만 선택 가능)
   - **별점순 정렬**: 토글 버튼 (별점 높은 순으로 정렬)
3. 적용된 필터는 목록 위에 **칩(태그)** 형태로 표시됩니다.
4. 칩의 **X** 버튼을 클릭하면 해당 필터가 해제됩니다.

### 3-3. 찜하기

- 로그인한 Artist/Gallery 유저만 갤러리 카드 우측의 **하트 아이콘**을 사용할 수 있습니다.
- 하트를 클릭하면 즉시 빨간색으로 채워지며 찜 목록에 추가됩니다.
- 다시 클릭하면 찜이 해제됩니다.
- Admin 계정에는 찜 버튼이 표시되지 않습니다.

## 4. 갤러리 상세 (공통)

### 4-1. 기본 정보

- **상단 이미지 슬라이더**: 좌우 화살표로 갤러리 이미지를 넘길 수 있습니다. 이미지를 클릭하면 확대 보기(Lightbox)가 열립니다.
- **찜하기**: 우측 상단 하트 아이콘 (로그인한 Artist/Gallery 전용)
- **갤러리 정보**: 이름, 주소, 전화번호, 별점, 한줄 소개, 인스타그램, 이메일
- **상세 소개**: 갤러리에 대한 자세한 설명 (갤러리 오너만 수정 가능)

### 4-2. 진행 중인 공모

- 해당 갤러리에서 진행 중인 공모가 D-day와 함께 표시됩니다.
- 마감일이 7일 이내면 빨간색으로 강조됩니다.
- 클릭하면 공모 상세 페이지로 이동합니다.

### 4-3. 홍보 사진

- 전시가 종료된 공모에 대해 갤러리 오너가 업로드한 홍보 사진이 표시됩니다.
- 사진을 클릭하면 확대 보기가 가능합니다.

### 4-4. 리뷰

- 갤러리에 대한 리뷰 목록을 확인할 수 있습니다.
- 각 리뷰에는 별점, 작성자명(익명일 경우 "익명의 예술가 N"), 리뷰 내용이 표시됩니다.
- 리뷰에 이미지가 있으면 클릭하여 확대할 수 있습니다.

### 4-5. 이미지 확대 보기 (Lightbox)

- 이미지를 클릭하면 전체 화면 Lightbox가 열립니다.
- **좌우 화살표**로 이미지 전환, **터치 스와이프**로도 전환 가능합니다.
- **배경 클릭** 또는 **Esc 키**로 닫습니다.

## 5. 모집 공고 (공통)

### 5-1. 공모 목록

- **마감일이 남은(D-day > 0) 공모만** 표시됩니다.
- 각 카드에는 공모 제목, 갤러리명, 유형(개인전/단체전/아트페어), 모집 인원, 지역, D-day가 표시됩니다.
- D-day 7일 이내는 빨간색으로 강조됩니다.
- 카드 클릭 시 공모 상세 페이지로 이동합니다.
- 갤러리명을 클릭하면 해당 갤러리 상세 페이지로 이동합니다.

### 5-2. 필터

- **[필터]** 버튼으로 필터 패널을 열 수 있습니다.
- **전시 지역**: 서울, 경기 북부, 경기 남부, 대전, 부산
- **공모 유형**: 개인전, 단체전, 아트페어
- **갤러리 별점**: 3점 이상, 4점 이상
- 적용된 필터는 칩으로 표시되며, X 버튼으로 해제합니다.

### 5-3. 빠른 지원 (Artist 전용)

- Artist 유저에게만 카드 우측에 **화살표(Send) 아이콘**이 표시됩니다.
- 클릭하면 해당 공모에 즉시 지원됩니다.
- 지원 시 마이페이지에 등록한 포트폴리오가 갤러리에 전송됩니다.

## 6. 공모 상세 (공통)

### 6-1. 상세 정보

- 공모 제목, 소속 갤러리, 유형, 모집 인원, 지역, D-day
- 공모 기간 (시작일 ~ 마감일)
- 전시 기간 (시작일 ~ 종료일)
- 공모 소개 텍스트
- 대표 이미지 (클릭 시 확대 가능)

### 6-2. 지원하기 (Artist 전용)

- **[지원하기]** 버튼을 클릭하면 포트폴리오가 갤러리에 전송됩니다.
- 이미 지원한 공모에는 중복 지원이 불가합니다.

### 6-3. 찜하기

- 하트 아이콘으로 찜 추가/해제 (Artist/Gallery 전용)

### 6-4. 홍보 사진

- 전시 종료 후 갤러리 오너가 업로드한 홍보 사진이 표시됩니다.
- 클릭 시 확대 보기가 가능합니다.

### 6-5. 삭제

- 갤러리 오너 또는 Admin만 **[삭제]** 버튼이 표시됩니다.
- 삭제 전 확인 대화상자가 나타납니다.

## 7. 혜택 (공통)

- Admin이 등록한 혜택 정보가 카드 형태로 나열됩니다.
- 각 혜택에는 제목, 설명, 이미지가 표시됩니다.
- **[자세히 보기]** 링크가 있는 경우 클릭하면 외부 페이지가 새 탭에서 열립니다.

## 8. Artist 유저 가이드

Artist는 갤러리를 찾고, 공모에 지원하며, 리뷰를 작성하는 역할입니다.

### 8-1. 마이페이지 탭 구성

로그인 후 마이페이지에 접속하면 5개의 탭이 표시됩니다:

| 탭 | 기능 |
|----|------|
| 프로필 | 프로필 사진 변경 안내 |
| 포트폴리오 | 전시 이력, 약력, 작품 사진 관리 |
| 찜 목록 | 찜한 갤러리/공모 모아보기 |
| 내 리뷰 | 작성한 리뷰 모아보기 |
| 지원 내역 | 지원한 공모 내역 확인 |

### 8-2. 프로필 사진 변경

1. 마이페이지 상단의 프로필 카드에서 **동그란 프로필 이미지 위에 마우스를 올립니다**.
2. 카메라 아이콘이 나타나면 클릭합니다.
3. 파일 선택 창에서 이미지를 선택하면 즉시 변경됩니다.

### 8-3. 포트폴리오 관리

포트폴리오는 공모 지원 시 갤러리에 자동 전송되는 중요한 자료입니다.

**약력 및 전시 이력 수정:**
1. **[포트폴리오]** 탭 선택
2. 우측 상단 **[수정]** 클릭
3. **작가 약력**과 **전시 참가 이력**을 입력합니다.
4. **[저장]** 클릭

**작품 사진 관리:**
- 최대 **30장**까지 업로드 가능합니다.
- **[+]** 버튼으로 이미지를 추가합니다 (여러 장 동시 선택 가능).
- 각 이미지의 **X** 버튼으로 삭제합니다.
- 현재 업로드된 장 수가 "(N/30)" 형태로 표시됩니다.

### 8-4. 찜 목록

1. **[찜 목록]** 탭 선택
2. 상단 필터로 **전체 / 갤러리 / 공모**를 구분하여 볼 수 있습니다.
3. 각 항목 표시 형태:
   - **갤러리**: 갤러리명
   - **공모**: "갤러리명 - 공모 제목"
4. 항목을 클릭하면 해당 갤러리/공모 상세 페이지로 이동합니다.
5. 우측 **X** 버튼으로 찜을 즉시 해제할 수 있습니다.

### 8-5. 리뷰 작성

갤러리 상세 페이지에서 리뷰를 작성할 수 있습니다.

1. 갤러리 상세 페이지 하단의 **리뷰 섹션**으로 이동합니다.
2. **별점**(1~5)을 클릭하여 선택합니다.
3. 리뷰 **텍스트**를 입력합니다.
4. (선택) **이미지**를 업로드합니다.
5. (선택) **익명 작성**을 체크하면 "익명의 예술가 N"으로 표시됩니다.
6. **[리뷰 등록]** 클릭

**리뷰 수정:**
- 마이페이지 **[내 리뷰]** 탭에서 연필 아이콘을 클릭하면 해당 갤러리 페이지로 이동합니다.
- 갤러리 상세 페이지의 리뷰 섹션에서 본인 리뷰의 **[수정]** 버튼을 클릭합니다.

**리뷰 삭제:**
- 마이페이지 **[내 리뷰]** 탭에서 휴지통 아이콘을 클릭합니다.
- 확인 대화상자에서 **확인**을 누르면 삭제됩니다.

### 8-6. 공모 지원

**목록에서 빠른 지원:**
1. 모집 공고 페이지에서 원하는 공모 카드의 우측 **화살표 아이콘**을 클릭합니다.
2. "지원이 완료되었습니다!" 메시지가 표시됩니다.

**상세 페이지에서 지원:**
1. 공모 상세 페이지에서 **[지원하기]** 버튼을 클릭합니다.

**지원 시 참고사항:**
- 포트폴리오(약력, 전시 이력, 작품 사진)가 갤러리 이메일로 전송됩니다.
- **포트폴리오를 미리 작성해두세요!** 빈 포트폴리오가 전송되면 불리할 수 있습니다.
- 동일 공모에 중복 지원은 불가합니다.

### 8-7. 지원 내역 확인

1. 마이페이지 **[지원 내역]** 탭 선택
2. 지원한 공모의 제목, 갤러리명, 지원일이 표시됩니다.

## 9. Gallery 유저 가이드

Gallery 유저는 갤러리를 등록하고, 공모를 생성하여 아티스트를 모집하는 역할입니다.

### 9-1. 마이페이지 탭 구성

| 탭 | 기능 |
|----|------|
| 프로필 | 프로필 사진 변경 안내 |
| 내 갤러리 | 갤러리 등록 및 상태 확인 |
| 내 공모 | 공모 등록 및 상태 확인 |

### 9-2. 갤러리 등록

1. **[내 갤러리]** 탭 선택
2. 우측 상단 **[갤러리 등록]** 버튼 클릭
3. 등록 폼이 열립니다. 아래 항목을 입력합니다:

   | 항목 | 필수 여부 | 설명 |
   |------|----------|------|
   | 갤러리명 | 필수 | 갤러리 이름 |
   | 주소 | 필수 | 갤러리 소재지 |
   | 대표자명 | 필수 | 갤러리 대표 이름 |
   | 전화번호 | 필수 | 연락처 |
   | 인스타그램 주소 | 선택 | Instagram 프로필 URL |
   | 이메일 주소 | 선택 | 갤러리 이메일 |
   | 지역 | 필수 | 서울/경기북부/경기남부/대전/부산 중 선택 |
   | 한줄 소개 | 필수 | 간단한 갤러리 소개 |
   | 대표 이미지 | 선택 | 갤러리 대표 사진 업로드 |

4. **약관 동의**: 폼 하단에 약관 내용이 표시됩니다. 내용을 확인한 후 **"위 약관에 동의합니다"** 체크박스를 체크해야 등록 요청 버튼이 활성화됩니다.
5. **[등록 요청]** 클릭 (약관 동의 필수)
6. "갤러리 등록 요청이 제출되었습니다." 메시지가 표시됩니다.
7. Admin이 승인할 때까지 **승인 대기** 상태입니다.

### 9-3. 갤러리 상태 확인

등록한 갤러리의 상태가 색상 태그로 표시됩니다:

| 상태 | 색상 | 의미 |
|------|------|------|
| 승인 대기 | 노란색 | Admin이 아직 검토하지 않음 |
| 승인 완료 | 초록색 | 검색에 노출되며 공모 등록 가능 |
| 승인 거절 | 빨간색 | 거절됨 (거절 사유 확인 가능) |

- 승인 거절된 경우 **"거절 사유:"** 텍스트가 빨간색으로 표시됩니다.
- 거절 사유를 확인한 후 정보를 수정하여 다시 등록할 수 있습니다.

### 9-4. 갤러리 상세소개 수정

갤러리가 승인된 후에는 상세 소개를 직접 수정할 수 있습니다.

1. 갤러리 상세 페이지로 이동합니다.
2. 상세 소개 영역의 **[수정]** 버튼을 클릭합니다 (갤러리 오너에게만 표시).
3. 내용을 수정한 후 **[저장]** 을 클릭합니다.

### 9-5. 공모 등록

**승인된 갤러리가 있어야** 공모를 등록할 수 있습니다.

1. **[내 공모]** 탭 선택
2. 우측 상단 **[공모 등록]** 버튼 클릭
3. 등록 폼이 열립니다:

   | 항목 | 필수 여부 | 설명 |
   |------|----------|------|
   | 갤러리 선택 | 필수 | 승인된 갤러리 중 선택 |
   | 공모 제목 | 필수 | 공모 이름 |
   | 유형 | 필수 | 개인전 / 단체전 / 아트페어 |
   | 모집 작가 수 | 필수 | 모집할 인원 수 |
   | 공모 시작일 | 선택 | 공모 접수 시작일 |
   | 공모 마감일 | 필수 | 공모 접수 마감일 |
   | 전시 시작일 | 선택 | 전시 시작 날짜 |
   | 전시 종료일 | 필수 | 전시 종료 날짜 |
   | 지역 | 필수 | 전시 장소 지역 |
   | 간단 소개 | 필수 | 공모 설명 |
   | 대표 이미지 | 선택 | 공모 대표 사진 |

4. **약관 동의**: 폼 하단에 약관 내용이 표시됩니다. **"위 약관에 동의합니다"** 체크를 해야 등록 요청 버튼이 활성화됩니다.
5. **[등록 요청]** 클릭 (약관 동의 필수)
6. Admin 승인 후 모집 공고 페이지에 노출됩니다.

### 9-6. 공모 상태 확인 및 관리

- 등록한 공모의 상태가 갤러리와 동일한 색상 태그로 표시됩니다.
- 공모 카드를 클릭하면 공모 상세 페이지로 이동합니다.
- 공모 삭제: 각 공모 카드의 **휴지통 아이콘**을 클릭합니다 (삭제 확인 대화상자 표시).

### 9-7. 홍보 사진 등록

전시가 종료된 후 홍보 사진을 업로드할 수 있습니다.

1. 갤러리 상세 페이지에서 종료된 전시를 찾습니다.
2. **[홍보 사진 추가]** 버튼을 클릭합니다.
3. 이미지를 업로드하고 캡션(설명)을 입력합니다.
4. **[업로드]** 를 클릭합니다.
5. 업로드된 홍보 사진은 갤러리 상세 페이지와 공모 상세 페이지에 표시됩니다.

## 10. Admin 유저 가이드

Admin은 플랫폼 전체를 운영하고 관리하는 역할입니다.

### 10-1. 마이페이지 탭 구성

| 탭 | 기능 |
|----|------|
| 프로필 | 프로필 사진 변경 안내 |
| 승인 관리 | 갤러리/공모 등록 승인 및 관리 |
| 히어로 관리 | 홈 히어로 슬라이드 관리 |
| 혜택 관리 | 혜택 정보 관리 |
| 이달의 갤러리 | 이달의 갤러리 선정 |

### 10-2. 승인 관리

#### 승인 대기 처리

1. **[승인 관리]** 탭 선택
2. **[승인 대기]** 서브탭을 선택합니다 (기본 선택됨).
3. 대기 중인 갤러리/공모 등록 요청 목록이 표시됩니다.
4. 각 요청에는 상세 정보가 함께 표시됩니다:
   - **갤러리 요청**: 이름, 주소, 전화번호, 대표자명, 지역, 인스타그램, 이메일, 대표 이미지, 소개
   - **공모 요청**: 갤러리명, 유형, 모집 인원, 공모 기간, 전시 기간, 대표 이미지, 소개
5. 승인/거절 결정:
   - **[승인]** 버튼: 즉시 승인됩니다. 갤러리는 검색에 노출, 공모는 공고에 노출됩니다.
   - **[거절]** 버튼: 거절 사유 입력 폼이 열립니다.
     - 거절 사유를 **반드시 작성**해야 합니다 (빈 사유로는 거절 불가).
     - **[거절 확인]** 을 클릭하면 거절이 완료됩니다.
     - Gallery 유저는 마이페이지에서 거절 사유를 확인할 수 있습니다.

#### 등록 관리 (삭제)

1. **[등록 관리]** 서브탭을 선택합니다.
2. 현재 등록된 모든 갤러리와 진행 중인 공모 목록이 표시됩니다.
3. 각 항목의 **휴지통 아이콘**을 클릭하면 삭제할 수 있습니다.
4. **주의**: 갤러리 삭제 시 관련 공모, 리뷰, 이미지 등 모든 데이터가 함께 삭제됩니다.

### 10-3. 히어로 슬라이드 관리

홈 화면 최상단 히어로 배너를 관리합니다.

**슬라이드 등록:**
1. **[히어로 관리]** 탭 선택
2. **[새 슬라이드]** 버튼 클릭
3. 정보 입력:
   - **제목** (필수): 배너에 표시될 제목
   - **설명** (선택): 부가 설명 텍스트
   - **링크 URL** (선택): "바로가기" 버튼 클릭 시 이동할 URL
   - **순서** (선택): 슬라이드 표시 순서 (숫자가 작을수록 먼저)
   - **이미지** (필수): 배너 이미지 업로드
4. **[미리보기]** 버튼으로 실제 배너 모양을 미리 확인할 수 있습니다.
5. **[등록]** 클릭

**슬라이드 수정:**
- 기존 슬라이드의 **연필 아이콘**을 클릭하면 수정 폼이 열립니다.
- 내용을 수정한 후 **[수정]** 을 클릭합니다.

**슬라이드 삭제:**
- 기존 슬라이드의 **휴지통 아이콘**을 클릭합니다.

### 10-4. 혜택 관리

혜택 페이지에 표시되는 혜택 정보를 관리합니다.

**혜택 등록:**
1. **[혜택 관리]** 탭 선택
2. **[새 혜택]** 버튼 클릭
3. 정보 입력:
   - **제목** (필수): 혜택 이름
   - **설명** (필수): 혜택 상세 설명
   - **링크 URL** (선택): "자세히 보기" 클릭 시 이동할 URL
   - **이미지** (선택): 혜택 대표 이미지
4. **[미리보기]** 버튼으로 실제 카드 모양을 확인할 수 있습니다.
5. **[등록]** 클릭

**혜택 수정/삭제:**
- 연필 아이콘(수정), 휴지통 아이콘(삭제)을 사용합니다.

### 10-5. 이달의 갤러리 관리

홈 화면에 표시되는 "이달의 갤러리"를 선정합니다.

**등록:**
1. **[이달의 갤러리]** 탭 선택
2. **[갤러리 선정]** 버튼 클릭
3. 검색창에 갤러리명을 입력하여 검색합니다 (이미 선정된 갤러리는 제외됨).
4. 검색 결과에서 원하는 갤러리를 클릭하여 선택합니다.
5. **만료일**을 설정합니다 (해당 날짜가 지나면 자동으로 제거됩니다).
6. **[등록]** 클릭

**삭제:**
- 현재 선정된 갤러리 목록에서 **휴지통 아이콘**을 클릭합니다.

**자동 만료:**
- 만료일이 지난 갤러리는 홈 화면 접속 시 자동으로 제거됩니다.

### 10-6. 리뷰 관리

- Admin은 갤러리 상세 페이지에서 모든 리뷰에 **[삭제]** 버튼이 표시됩니다.
- 부적절한 리뷰를 삭제할 수 있습니다.

### 10-7. 참고사항

- Admin 계정에는 **찜 기능이 비활성화**되어 있습니다 (운영 계정이므로).
- 승인 관리 페이지는 항상 최신 데이터를 가져옵니다 (캐시 없음).

## 11. 자주 묻는 질문 (FAQ)

### Q: 배포 사이트에 접속했는데 로딩이 오래 걸립니다.
**A:** 무료 호스팅(Render.com)을 사용하고 있어, 15분간 미사용 시 서버가 sleep 상태에 들어갑니다. 첫 접속 시 약 30초의 콜드스타트 시간이 필요합니다.

### Q: 배포 사이트에서 업데이트가 반영되지 않습니다.
**A:** PWA 캐시 때문입니다. 브라우저 DevTools(F12) -> Application 탭 -> "Clear site data" 클릭 후 새로고침하세요.

### Q: 갤러리를 등록했는데 검색에 안 나옵니다.
**A:** 갤러리 등록은 Admin 승인이 필요합니다. 마이페이지 [내 갤러리] 탭에서 승인 상태를 확인하세요. "승인 대기" 상태이면 Admin이 아직 처리하지 않은 것입니다.

### Q: 공모에 지원하려는데 버튼이 보이지 않습니다.
**A:** 지원하기 기능은 Artist 유저 전용입니다. Gallery 또는 Admin 계정으로는 지원할 수 없습니다. Artist 계정으로 로그인하세요.

### Q: 찜 버튼이 보이지 않습니다.
**A:** 두 가지 가능성이 있습니다:
1. 로그인하지 않은 상태 -> 로그인이 필요합니다.
2. Admin 계정으로 로그인 -> Admin에게는 찜 기능이 제공되지 않습니다.

### Q: 리뷰를 수정하고 싶습니다.
**A:** 마이페이지 [내 리뷰] 탭에서 연필 아이콘을 클릭하면 해당 갤러리 페이지로 이동합니다. 갤러리 상세 페이지의 리뷰 섹션에서 본인 리뷰의 [수정] 버튼을 사용하세요.

### Q: 공모 등록 시 "승인된 갤러리가 없습니다"라고 나옵니다.
**A:** 공모는 승인 완료된 갤러리에만 등록할 수 있습니다. 먼저 갤러리를 등록하고 Admin 승인을 받으세요.

### Q: 등록 요청 버튼이 비활성화되어 클릭할 수 없습니다.
**A:** 약관 동의 체크박스를 확인하세요. 폼 하단의 약관 내용을 읽고 "위 약관에 동의합니다" 체크박스를 체크해야 버튼이 활성화됩니다.

### Q: 갤러리 등록이 거절당했습니다. 어떻게 해야 하나요?
**A:** 마이페이지 [내 갤러리] 탭에서 거절 사유를 확인할 수 있습니다. 사유를 참고하여 정보를 보완한 후 다시 등록 요청하세요.

### Q: 포트폴리오는 언제 작성해야 하나요?
**A:** 공모 지원 시 포트폴리오가 갤러리에 전송되므로, **지원 전에 미리 작성해두는 것을 권장**합니다. 마이페이지 [포트폴리오] 탭에서 약력, 전시 이력, 작품 사진(최대 30장)을 등록하세요.
